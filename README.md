## Test Frontend (Sepolia)

A lightweight browser tester is available in the `frontend-test/` folder.

### How to run

1. Open `frontend-test/index.html` using a local static server (recommended), for example VS Code Live Server.
2. Connect MetaMask on Sepolia network.
3. Use the UI to:
	- load candidate vote counts
	- cast a vote by candidate index
	- refresh on-chain data
